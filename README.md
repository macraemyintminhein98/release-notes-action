# Release Notes Action

**Automate Your Release Notes.**

This GitHub Action automatically generates well-formatted release notes directly from your commit messages and pull request titles.

## Features:
-   Parses commit history and PRs.
-   Outputs in Markdown, JSON, Slack, and more.
-   Highly configurable to match your team's needs.

## Getting Started:
1.  Add the action to your GitHub Workflow.
2.  Configure your desired format.
3.  Enjoy automated, consistent release notes!
